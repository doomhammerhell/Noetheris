# Paper Artifacts

The Noetheris v0.1.0 technical note is maintained as Markdown at
`docs/paper/noetheris_v0_1_paper.md`.

It is part of the release evidence package and is intentionally scoped as a
research engineering note: it defines the model, executable artifacts,
validation boundaries, and reproducibility commands without claiming quantum
advantage, cryptographic compromise, or hardware performance.

The v0.2 working paper is maintained as LaTeX at
`docs/paper/noetheris_v0_2_short_paper.tex`. It narrows the next release around
the external solver boundary: canonical QUBO exchange, optional local Ocean and
Qiskit representations, and deterministic replay of solver candidates.
