# Ethical And Security Boundaries

Noetheris is defensive research infrastructure. It models structural risks and verifies bounded witnesses. It must not be presented as a system for attacking deployed cryptography or as evidence that production PQC deployments are compromised.

Every solver result is an untrusted candidate until deterministic certificate validation accepts it. Every certificate is relative to the correctness of the structural model and encoding. Model errors remain outside certificate guarantees.

Responsible extensions must state adversary assumptions, trust boundaries, replay strategy, and operational limits.
