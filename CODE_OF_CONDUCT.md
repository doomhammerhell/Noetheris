# Code of Conduct

Noetheris is a technical research project. Participants are expected to maintain a professional environment for rigorous engineering discussion.

## Expected Behavior

- Focus critique on technical correctness, evidence, and system properties.
- State assumptions explicitly.
- Treat security-sensitive topics with care and precision.
- Respect maintainers, contributors, and researchers across disciplines.

## Unacceptable Behavior

- Personal attacks, harassment, or discriminatory conduct.
- Publishing unsupported security claims under the project name.
- Using project channels to promote unsafe cryptographic exploitation.
- Deliberately submitting misleading benchmark or certificate results.

## Enforcement

Maintainers may remove content, reject contributions, restrict participation, or report abuse when conduct undermines the safety or integrity of the project.
